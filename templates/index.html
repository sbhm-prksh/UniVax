Vaccine Design Automation Tool

Follow the steps below to automate the design and testing process.

Step 1: Upload Your .fa File

Upload your FASTA file containing protein sequences. This file will be used for further processing and analysis.

Step 2: Convert .fa to .csv

Once the file is uploaded, it will be converted into a CSV format for easier processing and further analysis.

Step 3: Run Allergenicity, Signal P, and Antigen Tests

Run allergenicity tests, Signal P detection, and antigen testing on the sequences. View the results after processing.

Step 4: Sequence Filtering

Filter the sequences after the allergenicity, antigen, and Signal P detection steps to refine the dataset.

Step 5: B-cell Epitope Prediction

Predict B-cell epitopes from the filtered sequences. Run allergenicity, antigen, and toxicity tests on them to obtain the final predicted epitopes.

Step 6: T-cell A1 Supertype Epitope Prediction

Perform epitope prediction for T-cell A1 supertype using the same filtering process as Step 5.

Step 7: T-cell B58 Supertype Epitope Prediction

Perform epitope prediction for T-cell B58 supertype with similar filtering as Step 5 and 6.

Step 8: Th-cell Epitope Prediction

Run epitope prediction for Th-cells and filter the sequences similarly as in the previous steps.

Step 9: Download All Epitopes

Once all tests and predictions are complete, download the final epitopes generated from all the steps.
